# DAATAN Core Document
_This document is the single source of truth for the DAATAN project._

## 1. Project Mission
Based on the available information, DAATAN appears to be a web application focused on **forecasting and predictions**. It likely involves users making predictions, which are then resolved and tracked on a leaderboard.

## 2. Technology Stack
- **Framework:** Next.js
- **Language:** TypeScript
- **Frontend:** React, Tailwind CSS
- **Backend:** Next.js API Routes, Node.js
- **Database:** PostgreSQL (using Prisma ORM)
- **Authentication:** NextAuth
- **Testing:** Vitest
- **Infrastructure:** Docker, Nginx

## 4. Operational Commands
- **Install Dependencies:** `npm install`
- **Run Dev Server:** `npm run dev`
- **Run Tests:** `npm run test`
- **Build for Production:** `npm run build`
- **Start Production Server:** `npm run start`
